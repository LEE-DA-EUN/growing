**Application description:**

저희는 현대인들이 바쁜 삶 속에서 자신을 좀 더 들여다보고, 여러 긍정적인 변화를 도모할 수 있도록 다양한 기능을 넣어 Application을 개발할 예정입니다. 

Application의 이름은“시나브로”로 정했으며, 이는 "시나브로"의 의미(모르는 사이의 조금씩)처럼 앱을 사용하면서 차곡차곡 긍정적인 변화를 만들어내기를 바라는 마음을 담았습니다. 

또한, “시나브로”가 순우리말인 만큼, Application의 전체 디자인을 순우리말로 할 계획이며, 이를 통해 Application 사용자들은 더 많은 순우리말을 접하고, 그 단어가 주는 따뜻함을 동시에 느낄 수 있게 될 것입니다.

추가 기능과 개발 상황들은 계속해서 README.md와 문서정리 등을 통해 업데이트하도록 하겠습니다.
